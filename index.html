¿Qué son las Bases de Datos?

Las bases de datos son sistemas utilizados para almacenar y gestionar información de manera estructurada. Se clasifican en:

Tenemos las bases de datos relacionales que se caracterizan por almacenar información en tablas estructuradas donde cada fila representa un registro y cada columna un campo específico de ese registro. Estas bases de datos utilizan un lenguaje de consulta estructurado (SQL) para manipular y recuperar datos, garantizando la integridad de los datos mediante el uso de relaciones y claves primarias.

Además, existen las bases de datos no relacionales que se destacan por su flexibilidad en el manejo de datos. Estas no requieren una estructura de tabla fija como las relacionales y se adaptan mejor a tipos de datos variados y a estructuras de datos más desordenadas o jerárquicas, como documentos JSON, almacenamiento de clave-valor, grafos y bases de datos en columnas.
